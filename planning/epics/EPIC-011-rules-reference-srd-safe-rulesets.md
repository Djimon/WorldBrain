# EPIC-011: Rules Reference & SRD-Safe Rulesets

## Goal

Support rules reference and lightweight ruleset plugins while respecting licensing boundaries.

## Scope

- Rule entities.
- Source/license metadata.
- User Homebrew import.
- Clearly licensed SRD basis data.
- Example SRD-based structure/rules plugin.
- DM screen dashboard concepts.
- Simple non-AI evaluations where useful.

## Out Of Scope

- Protected WotC content packs.
- D&D Beyond scraping.
- Full encounter simulator.
- AI rules reasoning in V1.

## Open Decisions

- Minimum SRD-safe example data.
- Rule entity schema.
- Import format requirements.
- Depth of simple V1 evaluations.
