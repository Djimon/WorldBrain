# EPIC-006: Session Mode, Visibility & Conditions

## Goal

Support live GM workflows with session pages, visibility rules, player-facing projections, and controlled reveal state.

## Scope

- Session pages.
- Section-level visibility for V1.
- JsonLogic condition storage.
- Visual condition builder.
- Session variables.
- Player view/export without player accounts.
- Capture inbox.
- Session undo for GM mistakes.

## Out Of Scope

- Player accounts.
- Cloud collaboration.
- Full app-wide undo/history.
- Arbitrary script execution.

## Open Decisions

- Minimal visual condition builder vocabulary.
- Session variable type system.
- Exact session undo action list.
- Player view transport: local display, export, or generated handout bundle.
